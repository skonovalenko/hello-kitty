# svittiko

A personal 3D collection of beloved characters, built with [Three.js](https://threejs.org/).

Each figure is handcrafted from geometric primitives — no external models or textures.

**Characters**
- Hello Kitty
- Tigger
- Otter

## Tech

- Pure vanilla JS + ES modules
- Three.js via CDN import map
- No build step

## Run locally

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Deploy

Deployed on [Vercel](https://vercel.com) as a static site. Push to `main` to trigger a deploy.
